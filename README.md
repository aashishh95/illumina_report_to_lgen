## illumina_report_to_lgen
Conversion of Illumina Report to Lgen file format

## Overview
This script converts an Illumina Genotype Top Strand Matrix (CSV format) into a PLINK-compatible LGEN file. It is specifically designed to handle the "Top Strand" format commonly generated by Illumina's GenomeStudio Final Reports.

The script performs the following transformations:

Parses a wide-format matrix (Samples as columns, SNPs as rows).

Converts missing data indicators (--) to the PLINK-standard missing genotype code (0 0).

Outputs a long-format (LGEN) file suitable for downstream merging with sequencing data or other SNP datasets.

#Prerequisites
Python: 3.x

Input Format: A comma-separated values (.csv) file where:

The first row is a header containing sample IDs.

The first column contains Marker (SNP) names.

The cells contain diploid genotypes (e.g., AA, AG, GG, or --).
